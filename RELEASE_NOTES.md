# Release Notes

## v0.1.0

首个开源发布版本。

### 包含内容

- Tauri 桌面应用。
- 题库管理、录题、预览、导出图片和设置页面。
- 支持单选题、多选题、简答论述题。
- 支持导入 JSON 题库。
- 支持每日一题日期选择和预览日历。
- 支持更改本地数据文件夹，重启后生效。

### 不包含内容

- 不包含商业题库数据。
- 不包含用户本地 SQLite 数据库。
- 不包含 OCR/PDF/Word 原始资料。

### Windows 使用方式

1. 下载 `yiyan-daily-question-generator-v0.1.0-windows-portable.zip`。
2. 解压压缩包。
3. 双击 `毅研每日一题生成器.exe`。
4. 在题库页导入自己的 JSON 题库。
